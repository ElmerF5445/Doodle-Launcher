INFORMATION
Keyboard Shortcuts ===
Ctrl + Q - Collapse / expand all categories
Ctrl + B - Open clock screen
Ctrl + Shift - Open / close quick search bar
Ctrl + Right Arrow - Open / close sidebar
Ctrl + Up Arrow - Go to top of page
Ctrl + Down Arrow - Go to bottom of page
Alt + Up Arrow - Switch to upper tab
Alt + Down Arrow - Switch to lower tab
Escape - Close any active subwindow
Ctrl + Space - Opens the "Add Item" subwindow (Shortcut Editor)
Alt + Up Arrow - Switch to upper category link (Home)
Alt + Down Arrow - Switch to lower category link (Home)
